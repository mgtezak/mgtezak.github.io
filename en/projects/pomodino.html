PomoDino Timer

Customizable time management web application.
Flask, JavaScript, HTML & CSS

The Pomodoro Technique is a way of structuring one's work time. Traditionally, one alternates between a 25 minute work session (or "Pomodoro"), and a 5 minute break. After four work sessions one takes a longer 20 minute break, then the entire process is repeated. I like structuring my work day in this way and there are lots of online tools that are designed for this purpose (e.g. the classic pomofocus.io). I thought it would be good idea to try to build such an app myself, mainly for the purpose of learning JavaScript. For now I didn't use any frontend framework like React or Vue. I thought it best to get to know JavaScript in its pure form, but I do intend to explore React in the future. A good friend of mine, Bruno Kalthoff did most of the design and styling of the app, which allowed me to focus on the functionality. Thank you, Bruno! The app also has a Flask backend. Currently, it isn't actually doing anything except render the page, but I'm planning on introducing some features which will require backend operations such as user authentication and database interactions. I had a lot of fun building this app and the coolest thing is: I'm actually using it now! However, I prefer to use my own custom settings: Three cycles of a 55 minutes work session followed by a short 5 minute break, then a 45 minute food or walk break.
