Australian
Weather Prediction

Interactive web app to predict the weather in real-time.
Scikit-Learn, Bokeh, Selenium, SHAP, Streamlit, Docker

This is the final project of my three-month Data Science Bootcamp at DataScientest. The goal was to determine the probability of rain at 49 locations in Australia based on very limited weather data from the previous day. The end product is an interactive Streamlit app with a web scraping function. At the push of a button, the app retrieves today's weather data from the web within seconds, processes it, and depending on the location, runs it through one of many pre-trained ML models. The model then predicts tomorrow's weather and even interprets its own result, in that it provides some information on how the individual variables (temperature, humidity, etc.) have influenced the given forecast one way or the other.

The app is currently publicly accessible on the platform render.com. Unfortunately, due to the size of the app – it includes an entire Chrome browser to enable web scraping with Selenium – and because I chose the free option, it takes about 30 seconds to restart the Docker container when loaded for the first time. However, after that, it operates smoothly and with little latency. The associated Code Repository can be found on GitHub and the Docker Image on DockerHub.
